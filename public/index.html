URL Shortener Microservice

Pass a url to get back a shortened URL in the JSON response.

Examples:

https://thekholm80-url-shortener.herokuapp.com/new/https://www.google.com

Returns:

{ 'original_url': 'https://www.google.com', 'short_url': 'https://thekholm80-url-shortener.herokuapp.com/2000'}

Usage:

https://thekholm80-url-shortener.herokuapp.com/2000

Will redirect to:

https://www.google.com
